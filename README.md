Il Camino Della Mamma — Repositorio inicial

Proyecto Java — Gestor de recetas y comandas para restaurante

Resúmen

Aplicación de escritorio con Java (Java 17+ recomendado) que maneja recetas (JSON/XML), persistencia en MySQL mediante JPA/Hibernate y una interfaz minimalista con JavaFX. 
Soporta 4 roles: Administrador (chef), Ayudante (ver recetas y comandas), Camarero (crear/editar comandas), y Cliente (vista lectura a través de QR). 
Colores principales: #ddaa68, #fde8ca, #ff4b311f.
